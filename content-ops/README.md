# ChinaFreeWeight daily content operations

This directory is the audit trail for the 08:30 Asia/Shanghai publishing run. The target is five original B2B blog articles in each of the 12 public locales, but quality overrides quantity: only documents that pass every gate are published.

`runs/YYYY-MM-DD/research.json` stores query evidence, competitors and content gaps. `drafts/index.json` stores the common QA documents. `qa-report.json` is generated by the shared private registry. Published pages must also be added through the site's existing `ContentEntity`/manifest system (and the native English resources system where applicable), so metadata, schema, sitemap and language routing continue to work.

Generated images require an internal provenance and rights record. They may illustrate products or engineering concepts, but may not be represented as a real factory, customer installation, test, certificate or project.
